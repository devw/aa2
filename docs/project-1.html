Putting words into a binary search tree

Introduction

You are a team of young engineers responsible for maintaining and enriching a database of words in different languages. This database will then be used during the machine learning analysis.

The first corpus of words is provided to you (words.txt). It contains just over 20,000 proper nouns and common nouns of the English language. Your first task will be to implement in Python a particular data structure allowing you to classify these words in alphabetical order and to ensure a fast search for a word in the corpus (O(log(n))). Your project manager gives you one week to complete this task.

In one week, a new corpus of words will be provided to you (words-big.txt). You will need to implement an insertion function in the existing tree. The insertion of a new element should have a time complexity equal to O(log(n)). However, this requires your data structure to be balanced (perfect binary tree).

def get_data(fName):

def get_data(url):
    from urllib.request import urlopen
    response = urlopen(url)
    return response.read().decode('utf-8').split("\n")

get_words function


def get_words(data):
    words = []
    for word in data:
        words.append(word.rstrip("\n"))
    return words
